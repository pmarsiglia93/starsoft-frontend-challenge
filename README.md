## Starsoft Frontend Challenge

### Site
https://starsoft-frontend-challenge-lovat.vercel.app/

### Requisitos
- Node.js 18+ (recomendado 20)
- npm 9+

### Instalação
```bash
git clone https://github.com/pmarsiglia93/starsoft-frontend-challenge.git
cd starsoft-frontend-challenge
npm install
```

### Rodar em desenvolvimento
```bash
npm run dev
```

Acesse: http://localhost:3000

### Testes

```bash
npm test
```

### Lint

```bash
npm run lint
```

### Build

```bash
npm run build
npm run start
```

### Docker

- Se a porta 3000 já estiver em uso, pare o npm run dev antes.

```bash
docker compose up --build
```

Acesse: http://localhost:3000
